Общая идея

Данные директивы работают напрямую с HTML элементами и их событиями, реализуя сугубо утилитарный функционал, не связанный напрямую с функционалом right-angled. Такой, например, как предотвращения всплывания событий.

Как подключить

Чтобы подключить данные директивы, необходимо импортировать модуль RTMiscModule из библиотеки right-angled. Также можно импортировать модуль RtModule, включающий в себя функционал всей библиотеки right-angled целиком.

Директива rtStopEvents

Данная директива принимает в качестве параметра строку или массив строк с названиями событий.

Директива подпишется на указанные события элемента и будет вызывать в обработчике метод stopPropagation события, предотвращая дальнейшую передачу событий.

В примере ниже вы можете увидеть, как элемент input, помеченный данной директивой, останавливает распространение события keypress если установлена галочка "stop keypress events propagation" и не дает родительскому элементу div обрабатывать данное событие.

Директива rtPreventDefaults

Данная директива принимает в качестве параметра строку или массив строк с названиями событий.

Директива подпишется на указанные события элемента и будет вызывать в обработчике метод preventDefault события, предотвращая выполнение действий по умолчанию для событий.

В примере ниже вы можете увидеть ссылку, клик по которой не ведет к переходу благодаря директиве rtPreventDefaults.

Директива rtSelectOnFocus

Данная директива используется на элементах input и выделяет все содержимое элемента при фокусе.

Директива rtFocusOnRender

Данная директива, по сути, представляет собой аналог HTML атрибута autofocus, но работает при динамической отрисовке.

Элемент input, помеченный данной директивой, автоматически получит фокус после того, как будет отрисован.
