Параметры rt-paged-list

rtMinPageSize = "<number>"

Позволяет указать значение для свойства minPageSize.

rtMaxPageSize = "<number>"

Позволяет указать значение для свойства maxPageSize.

rtDefaultPageSize = "<number>"

Позволяет указать значение для свойства defaultPageSize.
