Общая идея

Данные директивы работают напрямую с браузерными элементами и их событиями, реализуя сугубо утилитарный функционал, не связанный напрямую с функционалом right-angled. Такой, например, как предотвращения всплывания событий.

Как подключить

Чтобы подключить данные директивы, достаточно импортировать модуль RTMiscModule из библиотеки right-angled.

Так же основной модуль RtModule уже включает в себя RTMiscModule, поэтому можно импортировать и его.

Директива rtStopEvents

Данная директива принимает в качестве параметра строку или массив строк с названиями браузерных событий.

Директива подпишется на указанные события элемента и будет вызывать для нихevt.stopPropagation(), предотвращая дальнейшую передачу событий.

В примере ниже вы можете увидеть, как элемент input, помеченый данной директивой, останавливает распространение события keypress если установлена галочка "stop keypress events propagation" и не дает родительскому элементу div обрабатывать данные события.

Директива rtPreventDefaults

Данная директива принимает в качестве параметра строку или массив строк с названиями браузерных событий.

Директива подпишется на указанные события элемента и будет вызывать для нихevt.preventDefault(), предотвращая выполнение действий по умолчанию для событий.

В примере ниже вы можете увидеть ссылку, клик по которой не ведет к переходу благодаря директиве rtPreventDefaults.

Директива rtSelectOnFocus

Данная директива используется на элементах input и выделяет все содержимое элемента при фокусе.

Директива rtFocusOnRender

Данная директива, по сути, представляет собой аналог HTML аттрибута autofocus, но работает при динамической отрисовке.

Элемент input, помеченный данной директивой, автоматически получит фокус после того, как будет отрисован он сам и его вложенный контент.
